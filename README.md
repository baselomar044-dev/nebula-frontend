# Nebula AI - Frontend

Static frontend for Nebula AI. Deploy to **Vercel**.

## Setup

1. Edit `config.js` - set your backend Railway URL
2. Deploy to Vercel
3. Done!

## Configuration

Edit `config.js`:
```javascript
CONFIG.API_URL = 'https://your-backend.up.railway.app';
CONFIG.WS_URL = 'wss://your-backend.up.railway.app';
```

## Local Development

```bash
npx serve .
# or just open index.html in browser
```
